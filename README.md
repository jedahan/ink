# ink

A [boot](http://boot-clj.com/) template for making sketches with [quil](https://quil.info).

## Usage

    boot --dependencies boot-new/new new --template ink --name my-cool-sketch
    cd my-cool-sketch
    boot dev
    open [http://localhost:3000]
